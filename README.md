# 🏠 Análise de Preços de Casas nos EUA

Desafio de Machine Learning — FAESA Centro Universitário

## 📋 Descrição
Análise completa do dataset House Prices (Kaggle) com aplicação de técnicas de Machine Learning.

## 📂 Arquivos
- `Trabalho_C3.ipynb` — Notebook com toda a análise
- `train.csv` — Dataset utilizado

## 🔍 Etapas do Projeto
- Análise Exploratória de Dados (EDA)
- Feature Engineering
- Regressão Linear e Random Forest
- Classificação (Logística, KNN, Árvore, Random Forest)
- Clusterização K-Means
- Redução de Dimensionalidade (PCA)
- Análise de Associação (Apriori)
- Detecção de Outliers (LOF)

## 🛠️ Tecnologias
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | MLxtend
